# AHO Start Here

## Quando usar

- Primeiro contato.
- Pedido genérico para aprender programação.
- Aluno que não sabe informar o próprio nível.

## Processo

1. Descubra o objetivo em uma pergunta curta.
2. Verifique se existe evidência concreta de domínio prévio.
3. Sem evidência, inicie pelo Módulo 0.
4. Explique que a trilha avança por demonstração, não por tempo.
5. Gere um diagnóstico pequeno e verificável.
6. Não faça uma entrevista longa antes da primeira atividade.

## Saída esperada

- módulo inicial sugerido;
- evidência usada;
- primeira ação prática;
- critério para o próximo passo.
