# AHO Algorithm Representation

## Pré-requisito absoluto

Esta skill só deve ser carregada quando o Módulo 1 (Pensamento Computacional) estiver concluído. O aluno já deve dominar análise, decomposição, reconhecimento de padrões, abstração e generalização.

## Fluxo

1. A partir de um problema já analisado, peça que o aluno identifique entrada, processamento e saída.
2. Defina regras e restrições com o aluno.
3. Peça que identifique casos normais, limites e inválidos.
4. Solicite a escrita do algoritmo em linguagem natural (passos numerados).
5. Converta para pseudocódigo quando útil para clarificar estrutura.
6. Use fluxograma como ferramenta auxiliar, não obrigatória.
7. Realize teste de mesa com pelo menos 3 casos.
8. Rastreie estado e valores das variáveis passo a passo.
9. Verifique se o algoritmo termina e produz o resultado esperado.

## Limite

Não avance para Python enquanto o aluno não conseguir:
- Escrever um algoritmo completo em linguagem natural.
- Rastrear um algoritmo simples com teste de mesa.
- Explicar entrada, processamento e saída.

## Regras

- Pseudocódigo e teste de mesa têm prioridade prática.
- Fluxogramas são auxiliares. Não os exija como requisito de checkpoint.
- Use problemas do cotidiano (receitas, instruções, jogos simples).
- Nunca use sintaxe Python neste módulo.
