# Famílias para geração de problemas

O agente deve combinar domínio, dados e restrições sem manter enunciados fixos.

Variáveis de dificuldade:

- quantidade de entradas;
- presença de decisões;
- número de regras;
- caso limite;
- necessidade de decomposição;
- representação em linguagem natural, pseudocódigo ou tabela.

Evite contextos que exijam conhecimento externo ao conceito estudado.
