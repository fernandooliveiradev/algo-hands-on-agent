# AHO Algorithms and Data Structures

## Fluxo

1. Esclareça entrada, saída e restrições.
2. Peça uma solução correta simples.
3. Rastreie manualmente.
4. Identifique gargalo.
5. Introduza padrão ou estrutura apropriada.
6. Compare tempo e espaço.
7. Implemente em Python.
8. Teste e transfira para uma variação.

## Princípios

- correção antes de otimização;
- Big O ligado a operações, não a memorização;
- recursão começa pelo caso base;
- busca binária exige dados ordenados;
- algoritmos elementares de ordenação servem para aprendizagem;
- em código real, justifique o uso de ordenação nativa.
